# Resume-Analyzer

This is a Python-based Resume Analyzer that reads a resume file and identifies skills present in it. It also suggests missing skills required for Data Analyst roles.

## Features 
- Extracts skills from resume
- Identifies missing skills
- Provides improvement suggestions

## Technologies Used 
- Python

## Sample Input 
A simple text file containing resume content.

## Output Example 
Skills found: Python, Excel
Missing Skills: SQL, Power BI, Tableau

## How to Run 
1. Clone the repository
2. Run the script:
          python main.py

## Project Purpose 
This project helps job seekers understand the gap between their current skills and industry requirements.
